## Notes

- **zsh** is not XDG compliant — `~/.zshenv` is unavoidable, used to set `ZDOTDIR` to redirect config into `~/.config/zsh/`
- **stow** is not XDG compliant — `.stowrc` must live in `$HOME` or the stow directory
